Spring-Grid K Regression Report (10x10 Physical, Damping + Bottom Forcing)

Setup

  • Grid: 10x10 (90 free DOFs after pinning the top row)
  • Springs: deterministic support-weighted lattice with the clamped support row at i = n-1 (top): vertical springs are slightly stiffer than horizontal ones, diagonals are softer braces, all springs stiffen near the clamped top row, and the centre of the panel has mild reinforcement
  • Mass M = I (unit point masses).
  • Damping C = alpha * M + beta * K with alpha = 0.050, beta = 0.020 (Rayleigh damping, treated as KNOWN in the regression).
  • Forcing: sinusoidal Gaussian point load applied along row 0 (bottom of plate). Each of the 4 trajectories uses the same amplitude (1.0) and Gaussian width (sigma = 1.5 columns) but a different angular frequency (multiples of base omega = 6.2832) and a slightly shifted column centre, so the data spans enough of the state space for K to be identifiable from zero initial conditions (treated as KNOWN in the regression).
  • Initial conditions: identically zero (no initial displacement, no initial velocity). The response is purely driven by the localized force.
  • Trajectories: 4, dt = 0.01, t_end = 20.0, total samples = 8004.
  • Train/test split: 6403 / 1601 samples (80/20 over pooled samples).
  • Regression target: target = a + C v - f, so that target = -K u. Only K is estimated; C and f are constants of the regression problem.
  • Recovered K = -Xi^T, symmetrised for physical diagnostics.
  • Locality: 8-connected neighbours (H + V + both diagonals) + self; 700 / 8100 (8.6%) of Xi entries are free parameters under the local mask.

Notes

The regression treats damping and forcing as KNOWN inputs. With M = I and the relation u_tt + C u_t = -K u + f, moving the known terms to the LHS yields (u_tt + C u_t - f) = -K u. We pass target = a + C v - f as the regression target so that the recovered Xi satisfies K = -Xi^T directly. STLS dense uses sequential thresholded least squares with no locality mask. SINDy Adam local zeros gradients on entries outside the 8-connected neighbour stencil.
