Spring-Grid Frequency Response Report (10x10, STLS Full + ROM)

Setup

  • Fine grid: 10x10 with 90 free DOFs (top row clamped).
  • ROM: block size 2, coarse grid 5x5, 25 DOFs.
  • Both regressed operators are fit by locality-masked STLS on the linear regression problem a + C v - f = -K u, using the same multi-frequency bottom-row training forcing as the rom-vs-full report.
  • Frequency response forcing is held fixed at unit amplitude on every bottom-row free DOF (in phase). The ROM forcing is the block-average restriction of this fine-grid forcing.
  • Mass: M = I. Damping: Rayleigh, C = alpha M + beta K_phys with alpha = 0.05, beta = 0.02. The same physical damping is paired with each predicted K when computing its response (damping is treated as known).
  • Steady-state response is computed analytically by solving (K - omega^2 M + i omega C) U = F0 at each omega.

Response curves

Three amplitude metrics over the same omega sweep. L2 is the Euclidean norm of |U(omega)| over all 90 fine free DOFs (ROM is lifted to fine via block-constant prolongation before measurement); max is the peak DOF amplitude; top-near is the mean amplitude on the row just below the clamped top. Vertical guides mark the true natural frequencies sqrt(eig(K_true)).

Reading the curves: the True model is plotted as a wide solid black line. The Regressed Full overlays it as a dashed blue line, and the Regressed ROM is a solid red line. When the Full regression converges to the true operator, the dashed blue line lies directly on top of the black True line. Quantitative deviation is shown in the dedicated error figure below.
